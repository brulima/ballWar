Ball War!

O Ball War! deve ser jogado com dois jogadores no mesmo computador.

Cada jogador escolhe uma cor (rosa ou azul). Bolinhas azuis e rosas aparecerão pela tela. Cada bolinha tem uma letra. O jogador que escolheu a cor azul deve apertar no teclado as letras das bolinhas azuis e o jogador que escolheu a cor rosa deve apertar no teclado as letras das bolinhas rosas.

Quando o jogador apertar a letra de uma bolinha, essa letra sumirá e irá aparecer uma nova bolinha com uma letra diferente. E o jogador deve continuar apertando as letras correspondentes às suas bolinhas

O jogador que fizer 1000 pontos ganha o jogo!

O Objetivo real do jogo é acabar em guerra. vale roubar o teclado e sair correndo. Vale cortar o dedo do amigo fora. Vale dedo no olho. Vale xingar a mãe.

O Ball War! não se responsabiliza por danos morais e físicos.

Atenção: para jogar é necessário estar com o teclado conectado.
